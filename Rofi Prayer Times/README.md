# Prayer Times Menu (Wofi)

## Install
./install.sh

## Run
prayer-times

## GNOME Shortcut
Settings → Keyboard → Custom Shortcuts
Command: prayer-times

## Requirements
- bash
- curl
- jq
- wofi

## Dependencies

### Ubuntu / Debian
sudo apt install curl jq wofi

### Arch
sudo pacman -S curl jq wofi

### Fedora
sudo dnf install curl jq wofi
